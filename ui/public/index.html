Estúdio · como funciona

Cada card é um tipo de vídeo (template) gerado pelos scripts gen-*.js e renderizado pelos render-*.js. Os arquivos vêm de output/videos/<tipo>/<slug>_<data>/ — onde imgs/ são os PNGs gerados, video/ os mp4 finais e audio/ as narrações.

  • Clique em um card para abrir o detalhe do tipo (slugs, vídeos, imgs).
  • Slug = um projeto/profissão dentro do tipo (ex: fisioterapeuta_2026-04-23).
  • Busca filtra por nome do tipo, slug ou arquivo de vídeo.
  • Read-only: a UI não dispara renders. Pra gerar novo vídeo, rode node gen-<tipo>.js <slug> no terminal.
